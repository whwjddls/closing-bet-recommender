# 종가베팅 추천 시스템 (closing-bet-recommender) — 설계안

- 작성일: 2026-06-30
- 상태: 설계 확정(브레인스토밍) → 스펙 리뷰 대기
- 유형: **추천/리서치 전용** (주문 실행 없음), **별도 신규 프로젝트**

---

## 0. 한 줄 요약

매 거래일 **15:20**(정규장 마감 15:30 직전)에, "종가베팅(신접갈거조재 + 시황 + 수급)" 방법론을 계산 가능한 신호로 번역하여 **다음날 상승 가능성이 높은 종목을 점수 순으로 최대 30개** 추천하는 대시보드. 오버나잇 효과(종가 매수 → 익일 시초/장 초반 매도)를 캡처한다. 자동 주문은 하지 않으며, **베테랑의 판단을 돕는 의사결정 지원 도구**로 설계한다.

---

## 1. 목표 / 비목표

### 목표
- 15:20 스냅샷으로 익일 상승 후보를 랭킹(최대 30개)하여 매수가/목표가/손절가/차트와 함께 제시
- 매일 추천을 자동 로깅하고 익일 결과로 채점하여 **적중률(성공률)** 추적
- 시황·수급·과열 등 리스크 오버레이로 "사면 안 되는 날/종목"을 걸러냄

### 비목표 (이번 범위 아님)
- 자동 주문/체결 (toss-autotrade가 담당, 책임 분리)
- 실시간 인트라데이 트레이딩(스캘핑) 신호
- 파생/해외/암호화폐
- 9개 신호 전부의 정밀 점수화 (YAGNI — 1차는 핵심만)

## 2. ⚠️ 전제와 솔직한 리스크 (설계 의사결정의 토대)

적대적 검증에서 확인된 핵심 — **반드시 기대치 설정에 반영한다.**

1. **9개 조건 → 실제 정보축은 3개.** 신·접·갈·조·거·재(긍정부)는 "오늘 거래량 터지며 고가 부근 강마감한 유동성 좋은 종목"이라는 **하나의 베팅의 다른 옷**. 단순 가중합은 한 요인을 6번 중복 투표시켜 가짜 합의를 만든다. 독립 정보축: ① 돌파강도 ② 시황 레짐 ③ 수급.
2. **단기 반전과 싸운다.** 종가베팅 픽은 한국(특히 코스닥)에서 가장 강건한 단기 효과인 **단기 반전(short-term reversal)** 에 역행한다. 인용되는 근거(52주 신고가=George-Hwang 2004, 거래량 프리미엄=Gervais-Kaniel-Mingelgrin)는 전부 **월간** 횡단면 효과이며, 이를 **1일 오버나잇**으로 옮긴 것은 검증되지 않았다. 매도세(0.18%)+호가스프레드+단일가 슬리피지 차감 후 순엣지는 ≤0일 수 있다.
3. **진짜 가치는 종목 찍기가 아니라 리스크 오버레이 + 청산 규율.** 시황 게이트, 희석(유상증자) veto, 유동성/과열 가드, 그리고 "보유 금지(익일 시초/초반 청산)" 규율이 좌측꼬리를 줄인다.
4. **신뢰 전 입증 의무.** 어떤 신호든 **순비용 차감 + 무작위/오버나잇-ETF 베이스라인 대비 incremental rank-IC** 를 보이기 전에는 가중치를 키우지 않는다.

> 결론: "알파 자동 발굴기"가 아니라 "검증된 리스크 필터 위에서 후보를 랭킹·시각화하는 의사결정 지원 도구". 백테스트는 낙관 편향(생존편향·룩어헤드·잠정→확정 갭)을 적극적으로 제거하는 방향으로만 신뢰한다.

## 3. 전략 → 계산 가능한 신호 (확정 MVP 셋)

### 3.1 채택/제외 요약

| 원 조건 | 처리 | MVP |
|---|---|---|
| **시황 (지수 5일선 레짐)** | HARD 게이트 (하락장이면 추천 축소/중단) | ✅ |
| **신 (52주 신고가 근접도)** | 돌파강도 **대표 점수** (연속 nearness) | ✅ |
| **거 (거래대금/거래량)** | 후보 풀 스크리너 + 임계 필터 (RVOL) | ✅ |
| **수급 (외인·기관)** | **D-1 확정치** 신뢰도 부스터(1.0~1.2x) | ✅ |
| **재 (재료)** | **희석/유상증자 공시 = 매수 veto** (긍정 재료 점수화는 제외) | ✅(veto만) |
| 위생 가드 | 유동성·우선주/ETF·관리/투자경고·과열(상한가/VI) 제외 | ✅ |
| 접 (전고점 이격) | 신과 collinear → **차트에 선만** | 시각만 |
| 갈 (깔끔한 양봉) | 캔들 알파 약함(Marshall 2006) → **차트 시각만** | 시각만 |
| 조 (조정 기간) | 9파라 트리거는 과최적 → "충분히 조정됨" **베이스 여부 보조표시** | 보조 |
| 섹터 주도성 | 1일 edge 얇음 → 2차 | 2차 |

### 3.2 신호별 정의 (계산 규칙)

**신 — 52주 신고가 근접도 (George-Hwang nearness)** [예측근거: 강함]
- `P_now` = 15:20 잠정 스냅샷가(KIS). `H_ref` = **전일까지 확정 EOD 일봉 고가**의 롤링 최대 (룩어헤드 금지).
- `nearness = P_now / H_ref`, 두 창 사용: N1=252거래일(52주), N2=60일(~3개월).
- 점수 `s_신 = 0.7·clip((near_252−0.90)/0.10,0,1) + 0.3·clip((near_60−0.90)/0.10,0,1)`.
- 적격 게이트: 이력 ≥120거래일, 20일 평균 거래대금 ≥ 10억, 우선주/SPAC/ETF/ETN/관리·투자경고·위험 제외.
- **과열 가드**: 당일 등락률 ≥ +20% 또는 상한가/VI 발동 → buy로 emit 금지(고점 매수=오버나잇 반전 위험).
- 비고: "신고가 돌파 ON CLOSE"는 15:20~15:30 단일가에서 뒤집힐 수 있으므로 **provisional 플래그**. 연속 nearness 비율을 신뢰.

**거 — 거래대금/RVOL** [예측근거: 보통, 비용취약]
- 후보 풀 = KIS 거래대금 상위 N(기본 200, 키움 0186 미러).
- `RVOL = 당일 누적 거래량(15:20 잠정) / 20일 평균 거래량`. 점수 `s_거 = clip(log2(RVOL)/log2(3), 0, 1)` (RVOL≥3 ≈ 만점).
- 과열 tier는 역전(너무 터지면 소진) → 과열 가드와 연동.

**시황 — 지수 5일선 레짐** [예측근거: 강건한 추세필터, 알파 아님]
- 코스피·코스닥 각각: 지수 종가(잠정) vs 5일 이평. 위/반등 = 매수 우호, 5일선 아래 반락 = 자제.
- 게이트 배수 `regime_mult ∈ {0, 0.5, 1}` (RISK_OFF면 추천 중단/축소).

**수급 — 외인·기관 (D-1 확정)** [예측근거: 진짜·직교, 단 modest]
- pykrx `get_market_trading_value_by_investor` 등으로 **전일 확정** 외인/기관 순매수. 당일치는 15:20에 공식 미확정(아래 §5).
- `supply_tilt ∈ [1.0, 1.2]` (외인/기관 D-1 순매수 강도에 비례).

**재 — 희석 veto (DART 공시)** [예측근거: veto가 핵심]
- DART OpenAPI로 유상증자/CB/BW/주식관련사채 등 **희석성 공시** 감지 시 `veto = 0` (추천 제외).
- 긍정 재료(테마/뉴스) 점수화는 백테스트 불가 → MVP 제외.

**위생 가드** — 유동성(거래대금), 종목타입(우선주/ETF/ETN/SPAC), 관리·투자경고·투자위험·단기과열 지정, 과열(상한가/VI) 제외.

### 3.3 복합 점수 (가중치 전부 백테스트 튜닝 전 잠정)
```
breakout = max(s_신, 0.5·s_접, 0.3·s_갈)      # 합산 금지 — collinearity 차단
base     = 0.5·breakout + 0.3·s_거 + 0.2·s_조
final    = base × regime_mult(0/0.5/1) × supply_tilt(1.0~1.2) × veto(0/1)
```
- 파이프라인: 위생필터 → 후보풀(거∪신) → HARD 게이트(시황·veto) → base → 부스터 멀티 → 정렬 → **상위 30**.
- 가중치/임계값은 walk-forward 튜닝 전 잠금; incremental rank-IC 입증 전 가중↑ 금지.

### 3.4 가격 규칙 (행마다)
- **매수가** = 15:20 잠정 스냅샷 (종가단일가 체결 목표, "잠정" 워터마크 표기)
- **목표가** = `매수가 + k·ATR20` (k=1.0~1.5) 또는 직전 전고점 중 가까운 값
- **손절가** = `매수가 − 1·ATR20` (또는 −3%)
- **청산 가정** = 익일 **시초가 / 09:30 VWAP** (보유 금지 권장 — 오버나잇 캡처)

## 4. 추천 산출물 & 성과 추적

### 4.1 추천 보드 (메인)
- 점수 높은 순 **최대 30개** 랭킹. 각 행: 종목명/코드, 현재가(잠정), 매수가, 목표가, 손절가, 점수/등급, 신호 배지(신고가·RVOL·수급·시황·베이스), 미니 차트, provisional 워터마크.

### 4.2 성과 추적 (적중률)
- **성공 정의(확정)**: **익일 시초가 수익률 > 0** (`open[t+1] / buy_price − 1 > 0`). 오버나잇 캡처 thesis와 정합.
- 매일 15:20 추천 스냅샷 저장 → 익일 시가 데이터로 자동 채점.
- 지표: 어제 N개 중 성공 M개(적중률 %), 평균 익일 시초 수익률, 누적 곡선, **점수구간별·시황레짐별 적중률**(어디서 잘 맞는지).
- 부가 표시(참고용): 목표가 도달 여부, 익일 종가 수익률.

## 5. 15:20 데이터 현실 (아키텍처 분기점)

| 구분 | 항목 | 처리 |
|---|---|---|

## 6. 아키텍처 (안 B: FastAPI + React/Vite)

### 6.1 구성
- **백엔드(Python)**: 추천 엔진 + 백테스트(순수 pandas) + 15:20 스케줄러.
- **데이터 레이어**: `pykrx`(EOD·D-1 수급·지수) + `KIS OpenAPI`(15:20 잠정 현재가/거래량/지수/과열) + `DART OpenAPI`(희석 veto). `BrokerDataAdapter` 추상화로 소스 교체 가능(현재 KIS).
- **API(FastAPI)**: `/recommendations/{date}`, `/stock/{code}`, `/performance`, `/backtest`, `/universe(scanner)`.
- **프론트(React+Vite)**: 차트 lightweight-charts / ECharts. (기존 toss-autotrade 대시보드 스택과 동일)
- **저장소**: `state/recommendations/YYYY-MM-DD.json`(추천 스냅샷) + **SQLite**(이력·채점·성과).

### 6.2 일일 흐름
```
15:20 스케줄러
  ① 후보 풀: KIS 거래대금 상위 N(200)
  ② 위생 필터: 유동성·종목타입·관리/경고·과열 제외
  ③ 시황 게이트: 코스피·코스닥 5일선 레짐 (RISK_OFF → 축소/중단)
  ④ 신호 계산: 신(52주 nearness)·거(RVOL)·수급(D-1)·재 veto
  ⑤ 복합 점수 → 정렬 → 상위 30 (매수가/목표가/손절)
  ⑥ 추천 스냅샷 저장
익일 → 어제 추천 자동 채점(익일 시가) → 적중률 갱신
```

### 6.3 화면
- 메인 추천 보드 / 종목 상세(일봉 + 52주고가 점선·전고점·베이스 박스·잠정 워터마크 + 신호 기여도) / 성과 추적 / 시황 레짐 게이지 / 후보 풀 스캐너(0186 미러)
- (2차) 섹터 히트맵 · 백테스트 대시보드 · 수급 패널 · 사람판단 보조 큐(당일 잠정수급·마감후 공시)

## 7. 안전 & 테스트

### 안전/정합성
- 추천 전용(주문 없음) → 자금 리스크 없음.
- **룩어헤드 방지**: `H_ref`는 전일까지로만 계산; 백테스트에서 당일 종가를 신호가로 쓰되 15:20 잠정과의 10분 갭을 명시.
- **생존편향 방지**: 상폐·관리·정리매매 종목 포함하여 백테스트 유니버스를 시점기준(point-in-time)으로 재구성.
- **분할 보정 누수**: FDR 수정주가의 사후 분할 반영 주의 → as-of-date 보정 또는 시점까지만 보정.
- **잠정 데이터 명시**: 모든 15:20 산출물에 provisional 표기.

### 테스트
- 신호 계산 단위테스트(임계값·경계), 채점 로직 테스트, 백테스트 룩어헤드 가드 테스트, KIS/DART 어댑터 목(Mock), 시황 게이트 레짐 분기 테스트.
- 백테스트는 **순비용 차감 + 오버나잇-ETF/무작위 베이스라인 대비 incremental rank-IC** 리포트를 필수 산출.

## 8. MVP 경계

- **1차(MVP)**: 시황 게이트 · 신(52주 nearness) · 거(RVOL) · 수급(D-1) · 재 veto · 위생필터 / 추천보드 · 종목상세 · 성과추적 · 레짐게이지 · 스캐너 / pykrx + KIS(+DART). 백테스트는 CLI 리포트 수준.
- **2차+**: 접·갈 점수화 · 조 정교화 · 섹터 · 백테스트 대시보드 · 당일 잠정수급 패널 · 사람판단 보조 큐.
- **제외**: 갈 독립알파, 거 EOD 프록시 라이브(룩어헤드), 9신호 단순 가중합, 긍정 재료 점수화.

## 9. 남은 미결 질문 (스펙 리뷰/계획 단계에서 확정)

1. **후보 풀 범위** — KIS 거래대금 상위 N(기본 200)으로 갈지, 전체 시장 스캔할지. (상위 N은 attention 반전·라이브 랭크 룩어헤드 위험 있음 — 백테스트 시 시점기준 유니버스 필요)
2. **KIS 커버리지** — 15:20에 현재가·누적거래량·지수·섹터 peer를 일괄 조회 가능한지, rate limit/배치 전략.
3. **백테스트 기간/데이터** — 대상 기간·체제(상승/하락/박스), 상폐·관리 리스트 확보 경로.
4. **운영 환경** — 로컬 PC 상시 실행(15:20 스케줄) vs 별도 호스트. PC 꺼져 있을 때 대비.
5. **종목 유니버스** — 코스피+코스닥 전체 vs 일부 제외(스팩·리츠 등 정책).
